Golden Meals - Restaurant Management Website
A professional, responsive, and interactive multi-page website developed for Golden Meals, a restaurant specializing in seafood and oriental cuisine. This project demonstrates proficiency in front-end development, data validation, and UI/UX design focused on user conversion.

🚀 Key Features
Interactive Menu: A dynamic meals list featuring "small-card" designs with an expand/collapse mechanism to view dish ingredients and details.

Smart Order Validation: A comprehensive ordering system implemented with JavaScript/jQuery that validates:

Arabic-only name inputs.

National ID (11-digit constraint).

Syrian mobile number formats (Syriatel/MTN).

Selection of at least one meal before proceeding.

Professional UI/UX: A clean, cohesive design using a color palette of Navy Blue and Orange, optimized for readability and branding.

Responsive Layout: Fully compatible with different screen sizes, ensuring a seamless experience on mobile and desktop devices.

Multi-Page Architecture: Includes Home, About Us, and Interactive Menu pages.

🛠️ Tech Stack
HTML5: Semantic structure for better SEO and accessibility.

CSS3: Custom styling with flexbox/grid and FontAwesome integration for icons.

JavaScript (ES6): Logic for data processing and input validation.

jQuery: Used for DOM manipulation and smooth UI transitions (Fade/Slide effects).

📂 Project Structure
home.html: Landing page featuring the development team and project overview.

aboutus.html: Information about the restaurant’s vision, features, and contact details.

meals.html: The core interactive menu and ordering form.

Style.css: Centralized stylesheet for consistent branding.

myfunction.js: Contains all the functional logic and validation scripts.
